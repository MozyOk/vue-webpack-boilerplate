vue-webpack-boilerplate
======================

A Webpack boilerplate with vue-loader,vue-resource and vue-router

## What's included
- Vue(2.x.x)
- webpack
- vue-loader
- vue-resource
- vue-router
- Bulma
- Font Awesome

## Setup
```
git clone -b gh-pages https://github.com/re-fort/vue-webpack-boilerplate
cd vue-webpack-boilerplate
npm i
npm run dev
```

## Usage
- `npm run dev`
- `npm run build`
